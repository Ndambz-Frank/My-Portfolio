# How to Add Your Resume PDF

## Steps to Add Your Resume

1. **Prepare your resume PDF**
   - Make sure your resume is in PDF format
   - Name it: `Francis_Muuo_Ndambuki_Resume.pdf`

2. **Place the file**
   - Navigate to the `public` folder in your project
   - Create a folder named `resume` if it doesn't exist
   - Place your PDF file in: `public/resume/Francis_Muuo_Ndambuki_Resume.pdf`

3. **Verify**
   - The resume will be automatically available for download
   - You can test it by running `npm start` and navigating to the Resume section

## File Structure
```
portfolio/
  public/
    resume/
      Francis_Muuo_Ndambuki_Resume.pdf
```

## Notes
- The file path in the code is: `/resume/Francis_Muuo_Ndambuki_Resume.pdf`
- Files in the `public` folder are served statically
- Make sure the filename matches exactly: `Francis_Muuo_Ndambuki_Resume.pdf`
